## Features

- **KPI cards** — total revenue, average ticket, total orders, goal attainment %, with YoY delta
- **Monthly revenue** — line chart with year-over-year comparison (2023 vs 2024)
- **Revenue by category** — sorted horizontal bar chart
- **Revenue by region** — vertical bar chart across Brazil's 5 regions
- **Top sellers** — bar chart with goal attainment % per seller (green = above goal, red = below)
- **Value distribution** — box plot by category showing spread and median ticket
- **Sidebar filters** — year, category, region, seller (all combinable)

## Project structure

```
sales-dashboard/
├── app.py                  # Streamlit entrypoint
├── generate_data.py        # Synthetic dataset generator
├── data/
│   └── sales.csv           # 8 000 transactions, 2023–2024
├── components/
│   ├── filters.py          # Sidebar filters
│   ├── kpis.py             # KPI metric cards
│   └── charts.py           # All chart functions (return go.Figure)
├── utils/
│   ├── data_loader.py      # CSV loader with @st.cache_data
│   └── formatters.py       # R$ / % / number formatting
├── styles/
│   └── theme.py            # Color palette and Plotly layout constants
├── .streamlit/
│   └── config.toml         # Dark theme configuration
└── requirements.txt
```

## Running locally

```bash
cd projects/sales-dashboard
pip install -r requirements.txt
streamlit run app.py
```

## Dataset

Synthetic data generated by `generate_data.py` — 8,000 transactions across:
- **5 categories:** Eletrônicos, Roupas & Calçados, Casa & Jardim, Alimentos & Bebidas, Esportes & Lazer
- **5 regions:** Sudeste, Sul, Nordeste, Centro-Oeste, Norte
- **8 sellers**
- **2 years:** 2023–2024, with seasonal spikes (Black Friday, Christmas, Dia dos Namorados)

## Deploying on Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Select the repo, branch `main`, entrypoint `projects/sales-dashboard/app.py`
4. Click **Deploy**
